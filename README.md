Quasar Extension for NeQu boilerplate
===

_Components and mixins for NeQu Boilerplate._


# Install
```bash
quasar ext add nequ
```
Quasar CLI will retrieve it from NPM and install the extension.

# Uninstall
```bash
quasar ext remove nequ
```
